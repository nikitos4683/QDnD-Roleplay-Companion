## 🌟 О приложении

**QD&D** (Q означает *Qurie*, создателя!) — это мобильное приложение для управления персонажами D&D 5-й редакции, созданное с абсолютной любовью к настольным ролевым играм. Забудьте о бумажных листах персонажей и бесконечных, запутанных таблицах — всё необходимое для игры теперь красиво организовано прямо в вашем кармане!

Будь вы закаленным в боях искателем приключений или начинающим героем, делающим свои первые шаги, QD&D станет вашим самым надежным спутником за игровым столом. Создавайте персонажей, управляйте заклинаниями, отслеживайте снаряжение и проводите бои с уникальными классовыми механиками — всё в одном невероятно сочном приложении в стиле Material 3 Expressive.

## ✨ Ключевые функции

### 🎨 Дизайн и персонализация
- **Темы:** Выбирайте из потрясающих цветовых схем (QMonokai, Gruvbox, Catppuccin и др.) со светлым/тёмным режимами.
- **Визуал:** Чистый дизайн Material 3 Expressive. Наслаждайтесь богатыми анимациями, вложенными поверхностями и приятной физикой движения при каждом взаимодействии.
- **Броски кубиков:** Встроенные 3D-анимации с физикой для приятных, тактильных бросков.

### 🎭 Управление персонажем
- **Мастер создания:** Пошаговое создание персонажа с мгновенным предпросмотром особенностей и интеграцией лора.
- **Мастер повышения уровня:** Интерактивное повышение уровня с умной поддержкой выбора (Подклассы, Боевые стили).
- **Универсальная система:** Полная, уникальная поддержка **ВСЕХ 12 ОСНОВНЫХ КЛАССОВ** благодаря мощной архитектуре, управляемой данными.
- **Импорт из Fight Club 5:** Бесшовный перенос ваших персонажей из XML.

### ⚔️ Боевая система
- **Трекер боя:** Инициатива, раунды и управление ходами на наглядной панели.
- **Уникальные панели классов:** Высокоинтерактивный, специально созданный интерфейс для каждого класса! Испытайте Ярость Варвара, Скрытность Плута (со смешными нарративными провалами), Кару Паладина, Ци Монаха и многое другое.
- **Живучесть:** Быстрое изменение HP с анимацией тряски, Спасброски от смерти и отслеживание временных HP.
- **Состояния:** Отслеживайте все состояния с подробными, легко читаемыми подсказками.

### 🔮 Магия и заклинания
- **Книга заклинаний:** Управление известными и подготовленными заклинаниями с умной фильтрацией.
- **Ячейки заклинаний:** Интерактивные трекеры, которые автоматически масштабируются с вашим уровнем.
- **Классовые особенности:** Умное отслеживание уникальных ресурсов (например, Очки чародейства, Вдохновение барда) и активных способностей.

### 🎒 Снаряжение и приключения
- **Экипировка:** Визуальные слоты для оружия/брони с автоматическим расчетом КД.
- **Кузница:** Мощный редактор создания предметов с предпросмотром в реальном времени, выбором иконок и визуальным редактором характеристик.
- **Журнал приключений:** Заметки с форматированием текста и отслеживанием квестов.
- **Оффлайн режим:** 100% работоспособность без подключения к интернету.

### 📚 Управление внешним контентом (DLC)
- **Менеджер библиотеки:** Импорт сторонних пакетов контента через XML (формат FC5).
- **Единая база данных:** Легкое объединение встроенного контента SRD с вашими собственными библиотеками (homebrew).
- **Двуязычность и умный парсинг:** Полная поддержка английского и русского языков. Приложение автоматически определяет и разделяет двуязычный текст в импортируемых файлах!

## 🚀 Начало работы

### 🤖 Установка на Android
1. **Скачайте последний APK** со [страницы релизов](https://github.com/QurieGLord/QDnD-Roleplay-Companion/releases).
2. **Установите** на ваше Android-устройство (Android 7.0+).
3. **Запустите** и начните свое приключение!

### 🍏 Установка на iOS (PWA через Safari)
*⚠️ Примечание: Полноценный нативный релиз для iOS пока недоступен. Однако вы можете использовать версию Progressive Web App (PWA)! Пожалуйста, имейте в виду, что стабильность PWA может отличаться по сравнению с нативным Android-приложением.*
1. Откройте Safari на вашем iPhone/iPad.
2. Перейдите к веб-приложению: **[Веб-версия QD&D](https://qurieglord.github.io/QDnD-Roleplay-Companion/)**
3. Нажмите кнопку **Поделиться** (квадрат со стрелкой вверх) в нижней части экрана.
4. Прокрутите вниз и выберите **"На экран «Домой»"**.
5. Нажмите **Добавить** в правом верхнем углу. Иконка QD&D появится на вашем домашнем экране и будет запускаться как нативное приложение!

## 🛠️ Сборка из исходного кода

```bash
# Клонировать репозиторий
git clone [https://github.com/QurieGLord/QDnD-Roleplay-Companion.git](https://github.com/QurieGLord/QDnD-Roleplay-Companion.git)

# Установить зависимости
flutter pub get

# Сгенерировать код (адаптеры Hive и локализация)
flutter pub run build_runner build --delete-conflicting-outputs
flutter gen-l10n

# Запустить
flutter run
```

### ✅ Недавние триумфы
- **v0.2.5 (The Wizard's Polish):** Значительные улучшения UI/UX в Мастерах создания персонажа и повышения уровня.

    - **Обзор M3 Expressive:** Полностью переработан экран обзора героя с вложенными карточками, структурированной сеткой внешности и красиво анимированными списками заклинаний.

    - **Выразительные селекторы выбора:** Все неудобные старые выпадающие списки заменены на премиальные, полностью локализованные модальные нижние панели в обоих мастерах.

    - **Умные заклинания и локализация:** Исправлено отображение предварительно заполненных заклинаний для подготавливающих заклинателей и создана глобальная архитектура локализации (D.R.Y.) для боевых стилей.

- **v0.2.0 (The Class Abilities Factory):** Масштабное визуальное обновление в стиле M3 Expressive. Все 12 базовых классов получили уникальные интерактивные панели (Механика Ярости, Механика Скрытности, Очки Ци и т.д.) с глобальной дедубликацией для безупречного пользовательского интерфейса.

### 🔮 Дальнейший путь
- **v0.5.0 (Many Paths):** Полная реализация системы МУЛЬТИКЛАССИРОВАНИЯ. Смешивайте и комбинируйте свои любимые классы с правильным масштабированием ячеек заклинаний и интеграцией особенностей.
